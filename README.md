# MS-OfficeProject

VBA-16. Sa se realizeze un proiect VBA care realizeaza importul unui fisier in format HTML si creaza pe baza acestuia un document Word cu formatari similare. Importul trebuie realizat la nivel jos, fara a apela la functia Open a mediului Word.

Elementele care trebuie avute in vedere:

formatarea fonturilor (culoarea fontului, bold, italic)
paragrafele, listele numerotate si nenumerotate
tabelele (doar tagurile de baza pentru definirea liniilor si celulelor: table, tr, td)
legaturile (hyperlinks)
Pentru toate tagurile corespunzatoare elementelor de mai sus nu se vor considera eventualele atribute. Exceptie fac legaturile si culoarea fontului, care pot fi definite doar ca atribute al unor taguri.

Se considera ca documentul HTML este corect (nu exista erori de sintaxa, taguri neinchise sau imbricate gresit). De asemenea, se considera ca in document nu exista liste imbricate (liste definite in interiorul altor liste) si nici liste definite in interiorul unor celule de tabel.

Nu este important modul de formatare a marginilor (borders) tabelelor.

Aplicatia gazda: Microsoft Word.

Numar de persoane: 3
